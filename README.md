# Biz Directory

Biz Directory is a full-stack web application built with Angular and Flask.
It allows users to browse businesses, read reviews and submit new reviews.

## Live Demo
Frontend:
https://mihaelaintech.github.io/biz-directory/

Backend API:
https://biz-directory-api.onrender.com/api/v1.0/businesses?pn=1

## Technologies
- Angular
- TypeScript
- Flask
- MongoDB
- Auth0
- GitHub Pages
- Render

## Features
- Business listing with pagination
- Individual business pages
- Reviews display
- Add review functionality
- Sentiment analysis on reviews
- User authentication with Auth0
